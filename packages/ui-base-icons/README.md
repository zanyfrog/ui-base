# @ui-base/icons

SVG icon registry and `uib-icon` Web Component.

## Embedded icon

```html
<uib-icon name="help" label="Help"></uib-icon>
```

## URL icon

```html
<uib-icon src="/assets/icons/calendar.svg" label="Calendar"></uib-icon>
```

## Decorative icon

```html
<uib-icon name="menu" decorative></uib-icon>
```

A decorative icon is hidden from assistive technology. A meaningful icon should receive `label`, `aria-label`, or `title`.
