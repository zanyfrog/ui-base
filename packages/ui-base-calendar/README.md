# @ui.base/calendar

Framework-neutral calendar Web Components for UI Base sites.

## Import

```js
import '@ui.base/calendar';
```

## Components

- `uib-calendar-day-view`
- `uib-calendar-week-view`
- `uib-calendar-month-view`
- `uib-calendar-year-view`
- `uib-date-window-view`
- `uib-day-of-week-view`

Each component is parent-state controlled and emits custom events for date, month, or page requests.
