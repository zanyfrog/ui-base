# @ui-base/design-system

Design standards and token definitions for the UI Base component ecosystem.

## CSS tokens

```html
<link rel="stylesheet" href="/node_modules/@ui-base/design-system/src/tokens.css">
```

Theme packages can load these tokens first and override selected values later. The later stylesheet wins because CSS custom properties cascade like normal CSS.
